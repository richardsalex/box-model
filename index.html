This page expects you to work in pixels (px); while the values will change even if you don't include a unit with your number, it's good practice to include it with all your values. Real CSS does not work this way.

Understanding the Box Model

We can think of many of the different content elements that we can add to a web page as existing in a rectangular box, including this example content division and the paragraph of text you're reading right now. The box model describes how margin, border, and padding work together to create space around and within that box.
